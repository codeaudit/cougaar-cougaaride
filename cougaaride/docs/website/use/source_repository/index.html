CougaarIDE Use:

Checking out a project from CVS as a Cougaar Project


Note, visit this link for tutorials on how to set up and use cvs repositories inside eclipse.

Select Window->Open Perspective->Other->CVS Repository Browser from the Eclipse workbench.  This will open the cvs repository perspective.



Expand the repository and Right Click on the module you would like to check and then select Check out as.



Select Check out as a project configured using the New Project Wizard and then press the Finish button.

Now follow the normal procedue for creating a new cougaar project.  See Creating a new Cougaar Project.

Note:  If the project you are checking out is a project that was created as a Cougaar Project in Eclipse and you have checked in the eclipse project files, you can select  Check out as a project in the workspace.  You will have to update the Cougaar Installation preferences once you have checked out the project.
