CougaarIDE -- An Integrated Development Environment for Cougaar

CougaarIDE is being developed by Cougaar Software Inc., to support developers interested in using Cougaar.

For Bugs, questions, feature requests, and other information, please visit the CougaarIDE Cougaarforge Project Page

CougaarIDE is comprised of a number of Plugins for Eclipse, you will need to install the Eclipse SDK before attempting to run CougaarIDE.
Currently CougaarIDE has been tested on Windows, MacOS X, and Red Hat Linux.
It should work with Eclipse Platform 2.1.x and the latest 3.0 stable build.

Look here for the current and upcoming features.
